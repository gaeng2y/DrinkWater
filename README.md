# 물마시자
현대인에게 부족한 수분 보충을 위해 매일 8잔을 마실 수 있도록 체크하기 위해 만들어 보았습니다.
매일 내가 마신 물의 양을 위젯을 통해 체크를 하자!

## 개요
하루 8잔 물을 마시면 건강해진다! 에서 아이디어를 얻어서 
위젯에는 내가 오늘 마신 물의 양을 표시해주고
앱에서 물을 마신 잔을 카운팅 하게 한다
